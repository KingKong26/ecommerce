## Getting Started

### Setup

* nodejs
* mongodb
* text-editor

### Installing
```
npm install
```
### Executing program

* Download or clone this repo into your local machine:
```
git clone https://github.com/KingKong26/ecommerce.git
```
*  Run:
```
npm start
```
* Setup environment variables in .env file in root directory
